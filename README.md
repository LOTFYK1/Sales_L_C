Project Overview

This project is an interactive Power BI dashboard consisting of three analytical report pages:

Sales Page – State-level sales analysis, top-selling products

Profit Page – Profit trends, profitability by state and product

Details Page – A drill-through view with detailed transactional insights

The dashboard analyzes retail performance from 2013 to 2016, helping identify trends and support decision-making.

Tools Used

Microsoft Power BI for data modeling, visualization, and KPI building
